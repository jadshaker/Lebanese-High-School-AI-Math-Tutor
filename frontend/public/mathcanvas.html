Math Canvas — Help Guide

Getting Started

Math Canvas is an interactive geometry and graphing tool powered by GeoGebra. Draw shapes, plot functions, measure distances and angles, and perform geometric constructions — all in your browser.

Basic workflow: Select a tool from the sidebar → click on the white canvas to place points and create objects → switch to Move to drag objects around. You can also type GeoGebra commands in the Command Bar at the bottom.

Drawing Tools

Command Bar

Type GeoGebra commands at the bottom and press Enter or click Run.

f(x) = sin(x) → plots a sine curve
g(x) = 2x + 1 → plots a straight line
Circle((0,0), 5) → circle at origin, radius 5
Polygon((0,0),(3,0),(3,3),(0,3)) → a square
Intersect(f, g) → find where two functions cross
